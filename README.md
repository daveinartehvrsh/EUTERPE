## setup
pip install all the libraries required:
* librosa
* numpy
* soundfile

## first run
After cloning the project to your local machine, go in the main folder and run

```
$ python run.py
```

## changing dataset
Dataset is located at data/loops.
Custom paths are to be set in preset.ini
